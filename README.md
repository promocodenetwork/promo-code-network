## 🔬 Verification Methodology

Every code listed here is verified through the following process:

1. **Source from operator directly** – codes come from the brand's affiliate program (Impact, Awin, CJ) or partner portal. Never from aggregator sites like RetailMeNot, Coupons.com, or Honey.
2. **Test on a real account** – every code is activated by signing up through the verified link and confirming the bonus applies as advertised.
3. **Document terms** – bonus amount, eligible region, expiration window, and KYC requirements recorded with verification date.
4. **Publish with verification date** – each page shows "Last verified: YYYY-MM-DD".
5. **Re-verify monthly** – every active code is re-tested at least once per month. Dead codes are removed within 24 hours of detection.
